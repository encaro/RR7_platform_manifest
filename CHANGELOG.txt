 ▪ d0f0018 f2fs-tools: disable clang on all targets
 ▪ 8460d9b fsck.f2fs: allow to disable forced full file system scans f2fs-tools keeps track of the kernel version by saving the content of /proc/version and forces a full file system scan when this changes from one boot to another. Since /proc/version includes a build timestamp, the scan is performed every time the kernel is re-built. Allow to disable the forced scan setting the environment variable F2FS_DISABLE_FORCED_CHECK to true when building fsck.f2fs.
